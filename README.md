# Habashy Python Automation Tools 🚀

هذه القائمة من الأدوات البرمجية مخصصة لأتمتة عمليات نقل الفيديوهات، ضغطها، ورفعها على Google Drive و Telegram.

## 📂 المكونات الرئيسية (Main Components)

1. **compressUpload.py**: 
   - يقوم بمراقبة مجلد الفيديوهات باستمرار.
   - يضغط الفيديوهات الجديدة باستخدام HandBrake.
   - يرفعها تلقائياً على Google Drive أو Telegram.
   - يرسل إشعارات لحظية بالتقدم لمجموعة التيليجرام.

2. **copy_mp3_auphonic_excel_download.py**:
   - يقوم بسحب الوسائط (صوت وفيديو) من كروت الذاكرة (Memory Cards).
   - يقوم بمعالجة الصوت باستخدام Auphonic لتحسين جودته.
   - ينظم الملفات في مجلدات يومية مرتبة.

3. **settings.xlsx**:
   - ملف الإعدادات الخاص بحسابات Auphonic والبيانات المرجعية.

## 🛠️ كيف تستخدم الكود على كمبيوتر جديد؟ (Getting Started)

1. **تثبيت بايثون**: تأكد من تثبيت Python 3.10 أو أحدث.
2. **تثبيت المكتبات**: افتح شاشة الأوامر (CMD) داخل المجلد واكتب:
   ```bash
   pip install -r requirements.txt
   ```
3. **تثبيت الأدوات المساعدة**: تأكد من وجود مجلد `bin` يحتوي على `ffmpeg.exe` و `HandBrakeCLI.exe` أو قم بتثبيتهم على النظام.
4. **التشغيل**: استخدم ملفات `run_...bat` لسهولة التشغيل بضغطة واحدة.

## ⚠️ ملاحظة هامة (Security Note)
تم إعداد ملف `.gitignore` لمنع رفع بياناتك الخاصة (مثل جلسات تيليجرام أو قواعد البيانات المحلية) للحفاظ على خصوصيتك. عند الرفع على GitHub ستنتقل الأكواد فقط دون البيانات الشخصية.
